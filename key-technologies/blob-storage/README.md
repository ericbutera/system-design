# Blob Storage

A way to store any sort of data.

- Distributed filesystem
- Replication (Durability)
- Multi mode storage (infrequent access in AWS glacier for lower cost)
- Versioning
- Encryption
- Permission based access
- Signed URLs (upload/download)
